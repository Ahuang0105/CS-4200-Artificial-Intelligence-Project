CS 4200 Project1

The IDE used to build this programs is Eclipse, the zip file "AlanHuang_4200p1.zip" file
contain 1 folder call "CS4200_Project_1". Please place the whole folder under your 
"eclipse- workspace" folder. Next open your Eclipse software and selete "File" at the very
top left corner, next click "open Projects from file System". Under the "import source" 
please selet the path to "CS4200_Project_1" folder and clip "finish". Also, the source
code is locate in the folder call "src" .


The file mainDrive.java has the main method that will test both of the heuristics for this project. 
After running the main method it will give you 3 options, random puzzle, manual input puzzle, 
or exit the program. Please selecte the option you want and hit enter.
