CS 4200 Project1

The IDE used to build this programs is Eclipse, the zip file "AlanHuang_4200p3.zip" file
contain 1 folder call "CS4200_Project_3". Please place the whole folder under your 
"eclipse- workspace" folder. Next open your Eclipse software and selete "File" at the very
top left corner, next click "open Projects from file System". Under the "import source" 
please selet the path to "CS4200_Project_3" folder and clip "finish". Also, the source
code is locate in the folder call "src" .


The file main.java has the main method that will test the 8x8 connect 4 game.
